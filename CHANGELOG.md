# 更新日志

本文档记录 TestCo-pilot 的所有版本更新。

## v0.1.0 (2026-06-15)

### 新增功能

- **AI 对话**：支持多种在线大模型接入
  - 小米 MiMo、阿里通义千问、DeepSeek、智谱 GLM
  - 百度文心一言、字节豆包、月之暗面 Kimi
  - MiniMax、零一万物 Yi、科大讯飞星火
  - OpenAI、Anthropic Claude
  - 自定义兼容 OpenAI API 的模型
- **本地知识库**：
  - 支持 TXT、Markdown、PDF、DOCX、XLSX 格式
  - 自动分块和向量化
  - 智能分类（刷写、诊断、通讯、网络管理等）
  - 递归扫描目录
- **测试报告分析**：
  - XML 报告解析
  - ZIP 文件解压
  - 失败用例提取
- **截图分析**：
  - Ctrl+V 粘贴剪贴板图片
  - 拖拽图片到窗口
  - 图片预览和删除
- **本地模型**：
  - 支持 GGUF 格式本地大模型
  - 需要 LLVM 编译环境（可选特性）
- **设置页面**：
  - 大模型服务商选择
  - 存储路径配置（目录选择器）
  - 字体样式和大小设置
  - 隐私与数据管理
  - 致谢声明
- **声明弹窗**：启动时显示免责声明
- **AI 对话原则**：注入系统提示词确保回答质量

### UI 设计

- 暖色奶油风设计语言
- 参考 MiModex 项目风格
- 小米橙色强调色
- 弹性动画效果
- 圆角卡片设计

### 技术栈

- Tauri 2.0 + React 19 + TypeScript
- Tailwind CSS 4 + Vite 7
- SQLite + reqwest + calamine

---

## 版本号说明

采用语义化版本：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
